Commandes utiles :

Créer projet -> composer create-project laravel/laravel "nom du projet"

consulter cache -> cache('nameOfKey')

Effacer cache -> cache()->forget('nameOfKey');

Démarrer serveur -> php artisan serve

php en ligne de commande -> php artisan tinker

créer une migration -> php artisan make:migration "nom" 

exécuter la migration -> php artisan migrate

Créer un controleur -> php artisan make:controller "nomController"
